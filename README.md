# DHMNN: A Hypergraph Motif-based Framework for Directed Hyperlink Prediction
The seven metabolic datasets are from http://bigg.ucsd.edu.

The one email dataset is from Reciprocity in ‘Directed Hypergraphs: Measures, Findings, and Generators’.

The candidate reaction pool for metabolic datasets is from ‘Beyond Link Prediction: Predicting Hyperlinks in Adjacency Space’.
